# Мини-документация по созданию формы с использованием `createForm` и `useAppForm`

## Описание

Наш API использует два основных элемента:

- **`useAppForm`** — хук для создания и управления состоянием формы. Принимает конфигурацию с `defaultValues`, `validators` и `onSubmit`.
- **`createForm`** — функция для создания React-компонента формы на основе описания полей и конфигурации опций.

---

## Шаги создания формы

### 1. Определение конфигурации формы

В конфигурации указываются:

- `defaultValues` — начальные значения для каждого поля.
- `validators` — объект с правилами валидации (например, можно использовать Zod-схемы).
- (не в `formOptions`! В `useAppForm`) `onSubmit` — функция для обработки отправки.
